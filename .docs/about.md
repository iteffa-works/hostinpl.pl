# About HOSTINPL

## What is HOSTINPL?

HOSTINPL is a billing and control panel for game and web hosting providers. It has been used in production since 2020 to sell and manage SAMP, MTA, Minecraft, Counter-Strike, RAGE:MP servers, and more.

## History

| Year | Event |
|------|-------|
| 2020 | Initial release by **HOSTINPL (HOSTING-RUS)** — Samir Shelenko, Alexander Zemlyanoy |
| 2020–2022 | Major 5.x feature set: payments, tickets, Docker locations, RAGE:MP, task scheduler |
| Feb 2022 | Debian 11 installer branch |
| 2026 | Open-source release on GitHub |
| 2026+ | **FLOWAXY DIGITAL STUDIO** takes ongoing development |
